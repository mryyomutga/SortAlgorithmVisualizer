# SortAlgorithmVisualizer
ソートアルゴリズムの可視化  
Visualize Sort Algorithm.  

## 搭載予定の機能
- ソートアルゴリズムの提供
- ソートをリアルタイムでビジュアライズ
- データに対応する音声出力

## 開発環境
- Processing 3.3.5

## 開発状況
- ランダムデータの生成
- データの値に対応するグラフの可視化
- バブルソートの実装
- 現在の比較データの可視化
- 440Hzの正弦波の音生成

## 発生しているバグ
- ソート完了時ArrayIndexOutOfBoundsExceptionが発生（ソートの完了動作未定義）
- バブルソートの動作を可視化できていない（挿入・ノーム + バブルソートのような動作になっている）
